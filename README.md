# Calorie-Tracker-A-Smart-Fitness-and-Nutrition-Management-Application
A Flutter-based fitness and nutrition tracking app that helps users monitor calories, macronutrients, water intake, workouts, and health progress with intuitive analytics and a user-friendly interface.
# 🥗 Calorie Tracker - Smart Fitness & Nutrition Management Application

A cross-platform mobile application developed using **Flutter** to help users monitor their daily calorie intake, nutritional values, water consumption, workout activities, and overall fitness progress.

The application provides an intuitive and user-friendly interface for maintaining a healthy lifestyle by tracking daily health activities and generating analytical reports.

## 👤 User Management

- User Registration
- Secure Login
- User Profile Management
- Logout Functionality

## 🍽️ Calorie Tracking

- Add Food Items

- Monitor Daily Nutrition

## 🥩 Nutrition Tracking

The application tracks essential nutrients including:

- Protein
- Carbohydrates
- Fat
- Fibre

This helps users maintain a balanced diet.

---

## 💧 Water Intake Tracker

Users can:

- Track Daily Water Consumption
- Monitor Hydration Progress
- Maintain Daily Water Goals

## 🏋️ Workout Tracking

The application allows users to:

- Record Workout Sessions
- View Workout History
- Estimate Calories Burned
- Track Daily Exercise

Generate detailed reports including:

- Daily Report
- Weekly Report
- Monthly Report

Reports help users analyze their fitness progress and nutritional habits over time.
